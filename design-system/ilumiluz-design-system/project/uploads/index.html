Sobre este documento

"Um sistema de design não é uma coleção de regras. É a gramática de uma linguagem visual."

Este documento define a linguagem visual completa da Ilumiluz — as decisões que tornam cada peça de comunicação reconhecível, coerente e alinhada com o posicionamento de ateliê contemporâneo.

A Ilumiluz vive entre dois mundos: a sofisticação atemporal da joalheria artesanal e a precisão contemporânea do design autoral com tecnologia. Este sistema reflete exatamente essa tensão.

Tipografia condensada e bold dialoga com serifas itálicas delicadas. Espaço generoso convive com pontos de cor intensa. Estrutura editorial encontra gesto artesanal. Cada decisão aqui foi tomada para sustentar essa dualidade.

Use este guia como referência para qualquer touchpoint da marca: site, Instagram, embalagem, comunicação — qualquer superfície onde a Ilumiluz se apresenta ao mundo.
