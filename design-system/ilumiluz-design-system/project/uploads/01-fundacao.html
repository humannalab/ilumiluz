Fundação
& Grid

Os princípios estruturais que sustentam toda a composição visual da Ilumiluz — espaço, proporção, respiro e intenção.

Princípios visuais

01

Espaço como valor

O vazio não é ausência — é intenção. Assim como o luxo da Ilumiluz está no tempo e na atenção, o espaço em branco é a respiração visual da marca. Nunca comprima. Sempre deixe a peça existir sozinha.

02

Assimetria elegante

Layouts levemente assimétricos criam tensão visual sofisticada. Evite o equilíbrio perfeito e previsível — prefira composições que surpreendem dentro de uma estrutura coerente.

03

Linha como detalhe

Bordas de 0.5px e divisores finos são a assinatura estrutural da Ilumiluz. Nunca use bordas espessas para separar conteúdo — a linha existe para guiar o olho, não para bloquear o fluxo.

04

90/10 — off white e cor

90% da superfície é sempre off white (#F7F3EC). Os 10% de cor são pontos de ênfase — ouro para calor e sofisticação, azul para profundidade e intenção. Nunca use cor de forma decorativa; use-a com propósito.

Escala de espaçamento

Tokens de Espaço

Escala multiplicadora baseada em 8px. Cada token tem um propósito específico — nunca use valores arbitrários.

--space-1
8px Gap interno mínimo, ícone-texto
--space-2
16px Padding interno de chips e tags
--space-3
24px Gap entre elementos próximos
--space-4
32px Margem entre título e corpo
--space-6
48px Padding lateral do site, margens de seção
--space-9
72px Padding vertical de seções
--space-12
96px Hero, seções de destaque
--space-20
160px Topo de páginas internas (compensa nav fixed)

Nenhum

0px

Mínimo

2px

Padrão

4px

A Ilumiluz usa bordas retas (0px) na maioria dos componentes. Apenas chips e tags usam 2px. Nunca use bordas arredondadas maiores — são inconsistentes com o posicionamento de marca.

Composição — Fazer e Evitar

✓ Fazer

Espaço generoso ao redor de elementos visuais. Deixe as joias e o texto respirar.

O luxo está no tempo, na atenção e no sentido.
✗ Evitar

Comprimir conteúdo, usar padding mínimo, empilhar elementos sem respiro visual.

✓ Fazer
